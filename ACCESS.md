# Access to the Zero-Shot Engineering Specification

This repository does **not** contain the full Zero-Shot Engineering (ZSE)
technical specification.

The canonical ZSE specification is distributed separately under
**controlled access**.

---

## What Is Access-Controlled

The following material is **not publicly available** in this repository:

- the full ZSE technical specification (PDF);
- normative definitions, formal constraints, and guarantees;
- complete operational protocols and case studies.

Only **descriptive, non-normative material** is hosted here.
